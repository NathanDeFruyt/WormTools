write.txt {WormTools}R Documentation

Write output to a .txt file

Description

Write your output to a .txt file

Usage

write.txt(data, file, suffix = ".txt", folder = getwd())

Arguments

data

The data you'd like to output. Make sure this is a character string.

file

The name you want your file to carry

suffix

By default: .txt .If you'd like to rather safe it as another file than .txt, take in mind the spacer then.

folder

The destination folder, by default: working directory

string.find {WormTools}R Documentation

find a substring in a string

Description

Similar to the str.find command in Python: find the index of a substring in a character string.

Usage

string.find(string, substring, mode = "first")

Arguments

string

The string you want to find it in

substring

The substring you want to find

mode

= c('first', 'all'): only the first occurence or all occurences?

Examples

string.find

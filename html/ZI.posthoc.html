Produce a post-hoc analysed data.frame for your fit.

Description

Produce a post-hoc analysed data.frame for your fit.

Usage

ZI.posthoc(fit, factors, nr.factors = 2, method = "pairwise",
  adjust = "tukey", selected.contrasts = F, contrasts)

Arguments

fit

The fit for which you'd like to produce a posthoc analysis. Don't forget to first run an Anova on it.

factors

The factors you'd like to check contrasts for. Give a vector containing all factors.

nr.factors

= 2 By default: nr.factor == 2. Chose c(1, 2, 3, 4)

method

= "pairwise" by default. Others are possible (check ?contrast)

adjust

= "tukey" by default. An adjustment of your p-values is usually required.
